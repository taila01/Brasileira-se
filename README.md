# 💻 Brasileira-se - Frontend (React)

Interface web moderna para visualização e cadastro de pontos turísticos do Brasil.

## Tecnologias
- **Framework:** React.js
- **Build Tool:** Vite
- **Estilização:** Tailwind CSS
- **Comunicação:** Axios (Consumindo a API na porta 5000)
- **Rotas:** React Router Dom

## Estrutura de Pastas (src)
- `components/`: Componentes reutilizáveis (`cards`, `form`, `layout`, `ui`).
- `pages/`: Telas da aplicação (`Home`, `Cadastro`, `Detalhes`).
- `services/`: Configuração do Axios para chamadas à API.
- `App.jsx`: Definição de rotas e estrutura principal.
